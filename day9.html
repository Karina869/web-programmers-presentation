Функции высшего порядка

Могут принимать другие функции как аргументы.

Могуг возвращаться другие функции как аргументы.

PHP позволяет писать программы в функциональной парадигме, но это совсем другая история.

Применение функций высшего порядка

function filter_array($arr, $filterFunct){
	$result = array();
	foreach($arr as $el) {
		if ($filterFunct($el)) $result[] = $el;
	}
	return $result;
}

Применение функций высшего порядка

		$filter1 = function($val){ return $val == 1;};
		$filter2 = function($val){ return $val == 2;};
		$filter3 = function($val){return $val != "."};
		var_dump(filter_array($arr, $filter1));
		var_dump(filter_array($arr, $filter2));
		var_dump(filter_array($arr, $filter3));

ООП

Программирование ориентированное на объекты

ООП - это не только программирование, но и проектирование

Объект - сущность наделенная состоянием и поведением.

Я - это объект. Моё поведение - это то, что я могу делать.

Моё состояние - это все мои характеристики, к примеру возраст или пол(надеюсь это константа).

Программа - это взаимодействие различных объектов.

Методы и свойства

Свойста(атрибуты, поля) - это переменные или константы, которыми обладает объект. Мои свойства - возраст, пол, семейное положение.

Методы - это функции объекта, то что объект может делать.

Кот Барсик - это объект. Его свойства или атрибуты - цвет и возраст. Его методы - есть, пить, мяукать.

ООП - это развитие процедурного стиля
